1. Contexto del proyecto



1.1. Nombre del proyecto.

Propuesta de sistema integrado para registro y control de asistencia del personal para Sistemas Aries SAS.

1.2. Nombre del sistema.

Sistema Integrado para Registro y Control de Asistencia del Personal (SIRCAP)

1.3. Antecedentes de la organización.

Sistemas Aries S.A.S tiene como actividad económica el desarrollo de sistemas informáticos (planificación, análisis, diseño, programación y pruebas de software) y su sector productivo es de Tecnologias de la información (TI) y servicios de tecnologías de la información (IT S services) por ende su actividad económica está relacionada con el desarrollo, implementación y mantenimiento de software y sistemas de información. Esta cuenta con varias áreas de gestión como: Área administrativa, Área de Desarrollo, Área de Soporte Técnico y Área de soluciones. Las tecnologías de la información y comunicación (TIC) que utilizan se dividen entre lenguajes de programación, entornos de desarrollo integrado (IDE), Base de datos, Frameworks, Servicios en la nube, central telefónica, software de gestión de tickets, software de escritorio remoto, base de datos de conocimiento y sistema de mensajería interna.

1.4. Antecedentes del sistema.

Autor/Año Nombre del sistema Ventajas Desventajas
Libro de control de asistencia Es simple y fácil de adquirir e implementar. Puede ser virtual (Excel) o físico. Al ser manual resulta poco preciso en cuanto a la veracidad y seguridad de los datos registrados.
NÜO / Blinda beep Tarjeta para control de acceso Es fácil de utilizar y genera un proceso más rápido y eficaz para el control en el acceso del personal. Es propensa a la pérdida, olvido (en casa) o daño y, facilita la posibilidad de falsificar el registro (individuo x utiliza la tarjeta de individuo y).
Bioentrada / GVS Colombia Huella y/o Biométrico El sistema es preciso y eficaz ya que el reconocimiento facial y dactilar es único en cada individuo, lo cual evita falsificación para el acceso. Puede ser un proceso costoso de implementar y mantener, además le impone al empleado brindar datos sensibles como lo es su identificación biométrica.
Sistemas Ares Aplicaciones móviles Permite una accesibilidad y gestión de la asistencia desde cualquier lugar (para el área encargada). Es fácil de implementar. Presenta una falta de precisión en los registros, los cuales también pueden alterarse, ya que los empleados podrían manipular la información con facilidad.
Salto wecosystem Pin / Clave Es preciso debido a que cada empleado cuenta con un número de PIN propio. Los empleados pueden olvidar su PIN, lo cual generaría inconvenientes su ingreso en la empresa.
